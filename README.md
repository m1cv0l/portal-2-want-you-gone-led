# Portal 2 — Want You Gone LED

Привет! Это мой небольшой проект, посвящённый к игре Portal 2 — **"Want You Gone"**.
Получилось атмосферно и очень в стиле Aperture Science.

### Что умеет проект:

- **Arduino Nano** управляет 8 оранжевыми LED
- Текст синхронизирован с песней
- На компьютере запускается красивая оранжевая консоль с логотипом Aperture Science и текстом песни

### Как запустить:

1. Загрузи прошивку `portal_led.ino` на Arduino Nano
2. Подключи 8 LED к пинам D2–D9 (через резисторы 220-330 Ом)
3. Запусти файл `start_portal.bat`

### Файлы в проекте:

- `Arduino/portal_led.ino` — прошивка для Arduino
- `Python/portal.py` — скрипт с логотипом и текстом песни
- `Python/start_portal.bat` — удобный запуск
